# RE/MAX טאבו נכסים — תבנית מודעות

## מטרת הפרויקט
כלי ליצירת מודעות נדל"ן לסוכן **משה שניאור** (RE/MAX טאבו נכסים).
משה **אינו טכנולוגי** — הכלי חייב להיות פשוט וידידותי לחלוטין.

## הקובץ הראשי
`מודעה_remax.html` — קובץ HTML יחיד, נפתח בדפדפן ישירות (Chrome/Safari).

### זרימת עבודה של משה
1. לחיצה כפולה על `מודעה_remax.html`
2. לחיצה על 4 ריבועי תמונה → בחירת תמונות מהמחשב
3. מילוי שדות: סוג נכס / פרטים / מחיר
4. לחיצה "צור מודעה (PNG)" → PNG מורד אוטומטית

## עיצוב — נעול, אל תשנה ללא אישור
| פרמטר | ערך |
|--------|------|
| צבע NAVY | `#1B3564` |
| צבע RED | `#CC0000` |
| פונט | Arial |
| כיוון | RTL (עברית) |
| רוחב מודעה | 520px (מוצג × 2 ב-PNG) |

## רכיבים נעולים (לא לשינוי)
- לוגו RE/MAX — מוטמע כ-base64 ב-HTML
- תמונת משה שניאור — מוטמעת כ-base64 ב-HTML
- שם הסוכן: **משה שניאור | זבין**
- טלפון: **052-8110611**
- Footer ו-Header — עיצוב קבוע

## שדות שמשה יכול לשנות
- סוג נכס (input חופשי)
- פרטים (textarea, מרובה שורות)
- מחיר (input חופשי)
- הצגת badge "עסקת השנה" (checkbox)

## אם צריך לעדכן תמונות (לוגו / תמונת משה)
1. המר את התמונה החדשה ל-base64: `base64 -i image.png | tr -d '\n'`
2. חפש בקובץ HTML את `data:image/png;base64,` הרלוונטי
3. החלף את ה-base64 הישן בחדש
